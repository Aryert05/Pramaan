## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aryert05/Pramaan.git
   cd Pramaan
   ```

2. **Install frontend dependencies**
   ```bash
   cd frontend
   npm install
   ```

3. **Install backend dependencies**
   ```bash
   cd ../backend
   npm install
   ```

## Usage

### Running the Frontend

```bash
cd frontend
npm start
```

The frontend application will typically run on `http://localhost:3000` (or another port if 3000 is occupied).

### Running the Backend

```bash
cd backend
npm start
```

The backend server will start on the configured port (typically `http://localhost:5000` or `http://localhost:8000`).

### Running Both Concurrently

From the root directory, you can set up scripts to run both frontend and backend simultaneously:

```bash
npm run dev
```

## Development

### Frontend Development

### Backend Development
